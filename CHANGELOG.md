# Changelog

All notable changes to the Trivx AI VS Code Extension will be documented in this file.

## [0.1.0] - 2024-01-01

### Added

- Initial release of Trivx AI VS Code Extension
- **Authentication**: Token-based and browser login with secure storage
- **Organization Management**: Switch organizations, view members and plan details
- **Project Management**: Create projects, link GitHub repos, browse in sidebar
- **Deployment**: One-click deploy with real-time Socket.IO progress tracking
- **AI Chat**: Full AI assistant panel with SSE streaming and code actions
- **CI/CD Pipelines**: View, trigger, and monitor pipeline runs
- **Secrets Management**: Create env groups, add/view/delete secrets
- **Monitoring**: Incidents, logs, metrics, and SRE dashboard
- **Status Bar**: Auth status, org/project context, deploy indicator
- **6 TreeViews**: Welcome, Organization, Projects, Deployments, Pipelines, Monitoring
- **18 Commands**: Full command palette integration
- **WebView Panels**: AI Chat and Deployment Progress with rich UI
